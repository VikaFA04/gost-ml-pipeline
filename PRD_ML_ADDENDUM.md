# Дополнение к PRD: PDF без OCR и NLP/ML

## PDF без OCR

Система поддерживает PDF только при наличии текстового слоя. OCR, сканированные PDF и изображения страниц не входят в MVP.

PDF-режим включает:

- извлечение текстовых блоков;
- извлечение базовых layout-признаков;
- классификацию блоков ML-моделью;
- формирование отчета нормоконтроля.

PDF не редактируется. Автокоррекция выполняется только для DOCX.

## NLP/ML

Система использует методы NLP для классификации структурных блоков документа. Основной рекомендуемый подход: TF-IDF-представление текста, форматные признаки и Linear SVM.

Rule-based слой, основанный на формализованных требованиях ГОСТ, является обязательным компонентом и не заменяется ML-моделью.
